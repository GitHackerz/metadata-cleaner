# Contributing

Thank you for your interest in contributing to Metadata Cleaner! We welcome contributions from the community.

## How to Contribute

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-feature-name`.
3. Make your changes and ensure tests pass: `cargo test`.
4. Commit your changes: `git commit -m "Add your message"`.
5. Push to your fork: `git push origin feature/your-feature-name`.
6. Open a Pull Request.

## Guidelines

- Follow Rust best practices and the existing code style.
- Add tests for new features.
- Update documentation as needed.
- Ensure your changes work on all supported platforms (Windows, macOS, Linux).

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub with as much detail as possible.

## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).
